   - **כותרת:** למשל "אני אוהב אותך"

   - **תאריך:** (השאר ריק כדי שהמתנה תופיע מיד)
     - או בחר תאריך עתידי → המתנה תופיע רק ביום הזה

## 🔧 תפעול שוטף

### עדכון האפליקציה:

1. ערוך את הקבצים המקומיים (`lumina-vault.html`, וכו')
2. Commit ו-Push ל-GitHub:
   ```bash
   git add .
   git commit -m "Update: [תיאור השינוי]"
   git push origin main
   ```
3. GitHub Pages יעדכן אוטומטית תוך 1-2 דקות
4. בטלפון: רענן את האפליקציה (pull down to refresh)

### גיבוי נתונים:

1. פתח את האפליקציה
2. לחץ על כפתור הגיבוי (אם יש)
3. או: העתק את הנתונים מ-localStorage דרך Developer Tools

### שחזור נתונים:

1. פתח את האפליקציה
2. לחץ על כפתור השחזור
3. או: הדבק נתונים ל-localStorage דרך Developer Tools

### פתרון בעיות:

**האפליקציה לא נטענת:**
- ודא שהכתובת נכונה
- בדוק את הקונסולה (F12) לשגיאות
- ודא ש-GitHub Pages מופעל

**הנתונים נעלמו:**
- בדוק אם localStorage נמחק (Settings → Clear Data)
- הנתונים נשמרים במכשיר בלבד
- אין גיבוי אוטומטי - צריך לגבות ידנית

**האייקון לא מופיע:**
- ודא ש-`assets/icon.png` קיים
- בדוק שהנתיב ב-`manifest.json` נכון
- נקה את ה-cache של הדפדפן
