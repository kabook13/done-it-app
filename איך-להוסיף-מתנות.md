# 🎁 איך להוסיף מתנות לאשתך

## ✅ הקוד כבר מוכן!

הפיצ'ר "מתנות ממני" כבר נוסף לאפליקציה. עכשיו צריך:

1. **לעדכן את האפליקציה ב-GitHub Pages**
2. **להוסיף מתנות דרך האפליקציה**

---

## 📤 שלב 1: עדכון האפליקציה ב-GitHub

### א. העלה את הקובץ המעודכן:

1. **פתח את GitHub** → היכנס ל-repository שלך (`done-it-app`)
2. **לחץ על `lumina-vault.html`** → לחץ על עיפרון (✏️) לעריכה
3. **העתק את כל התוכן** מהקובץ `lumina-vault.html` במחשב שלך
4. **הדבק ב-GitHub** (החלף את כל התוכן הישן)
5. **גלול למטה** → לחץ **"Commit changes"**
6. **חכה 1-2 דקות** → GitHub יעדכן את האתר

### ב. בדוק שהכל עובד:

1. **פתח את האפליקציה** בדפדפן: `https://YOUR_USERNAME.github.io/done-it-app/lumina-vault.html`
2. **תראה טאב חדש** "💝 ממני" בתפריט התחתון
3. **לחץ עליו** → תראה את המסך של המתנות

---

## 💝 שלב 2: הוספת מתנה ראשונה

### א. פתח את פאנל הניהול:

1. **פתח את טאב "💝 ממני"** באפליקציה
2. **לחץ 5 פעמים** על הכותרת "מתנות ממני" (בחלק העליון)
   - או לחץ על הכפתור הקטן למטה (🔧)
3. **תראה פאנל אדום** עם גבול סגול → זה פאנל הניהול!

### ב. הוסף מתנה:

1. **כותרת:** למשל "אני אוהב אותך"
2. **הודעה:** למשל "רק רציתי להזכיר לך כמה את מדהימה ❤️"
3. **אימוג'י:** למשל 💝 (או 🌹, 💌, ✨ וכו')
4. **תאריך:** (השאר ריק כדי שהמתנה תופיע מיד)
   - או בחר תאריך עתידי → המתנה תופיע רק ביום הזה
5. **לחץ "➕ הוסף מתנה"**

### ג. בדוק שהמתנה נוספה:

1. **סגור את פאנל הניהול** (לחץ "הסתר")
2. **תראה את המתנה** ברשימה
3. **לחץ עליה** → אנימציית confetti! 🎉
