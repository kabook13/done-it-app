# בקשת מחקר: חישובים סטטיסטיים למערכת אימון קוגניטיבי

## הקשר

אני מפתח מערכת אימון קוגניטיבי (brain training) דומה ל-Effectivate, Lumosity, BrainHQ. המערכת כוללת משחקי מוח שמאמנים תחומים קוגניטיביים שונים.

## המערכת הנוכחית

### משחקים:
1. **Go/No-Go** - אימון קשב ועיכבה
   - מדדים: דיוק, זמן תגובה, יציבות (CV)
   - חישוב נוכחי: דיוק(55%) + מהירות(30%) + יציבות(15%)

2. **Stroop** - אימון עיכבה ומהירות עיבוד
   - מדדים: דיוק, זמן תגובה ממוצע
   - חישוב נוכחי: דיוק * 100 (פשוט מדי)

3. **N-Back 1** - אימון זיכרון עבודה
   - מדדים: דיוק, זמן תגובה ממוצע
   - חישוב נוכחי: דיוק * 100 (פשוט מדי)

4. **Visual Search** - אימון קשב ויזואלי
   - מדדים: דיוק, זמן ממוצע למציאת מטרה
   - חישוב נוכחי: דיוק * גורם_מהירות * 100

### מבנה נתונים:
- כל ניסיון נשמר עם: `metrics` (JSON), `scores` (JSON), `domain_contrib` (JSON)
- סיכום יומי: `daily_score`, `domains` (JSON), `games` (JSON)

### תחומים קוגניטיביים:
- attention, inhibition, processing_speed, working_memory, reasoning_flexibility, visual_perception

2. **שילוב ציונים לא אופטימלי:**

3. **אין הדרגתיות אוטומטית:**

### 2. חישובים סטטיסטיים מתמטיים

**א. חישוב ציון למשחק בודד:**
- מה השיטה הנכונה לשלב דיוק, מהירות, ויציבות?
- איך לנרמל ציונים בין משחקים שונים?
- איך לטפל בנתונים חריגים (outliers)?
- איך לחשב ציון יחסי (percentile)?

**ב. שילוב ציונים מכמה משחקים:**
- איך לשלב ציונים מכמה משחקים לציון קטגוריה?
- איך לשלב ציוני קטגוריות לציון כולל?
- מה המשקלות הנכונים?
- איך לטפל במשחקים שלא שוחקו?

**ג. חישוב מגמות והתקדמות:**
- איך לחשב מגמה (trend) בצורה מדויקת?
- איך לזהות שיפור משמעותי סטטיסטית?
- איך לחשב "שיפור יחסי"?
- איך לטפל בתנודתיות טבעית?

**ד. נורמליזציה:**
- איך לנרמל ציונים לפי קושי המשחק?
- איך להשוות ביצועים בין תקופות שונות?
- איך לחשב "רמת כושר קוגניטיבי" כוללת?

### 3. הדרגתיות והתקדמות

**א. מתי להעלות קושי?**
- מה הקריטריונים להעלאת difficulty level?
- כמה ניסיונות צריך לפני העלאה?
- איך לטפל בירידה בביצועים?

**ב. מעקב אחרי שיפור:**
- איך למדוד שיפור לאורך זמן?
- איך לזהות "plateau" (עצירה בהתקדמות)?
- איך להמליץ על שינוי בתוכנית?

**ג. התאמה אישית:**
- איך להתאים קושי למשתמש ספציפי?
- איך לזהות תחומים חלשים ולחזק אותם?
- איך ליצור "תוכנית אימון" מותאמת?

### 4. חבילות אימון

**א. סוגי חבילות:**
- מה נהוג להציע? (יומי, שבועי, חודשי)
- כמה זמן אימון מומלץ ביום?
- כמה משחקים מומלץ לשחק ביום?

**ב. תוכניות אימון:**
- איך לבנות תוכנית אימון של שבוע/חודש?
- איך לשלב משחקים שונים?
- איך להגדיר יעדים?

**ג. תדירות אימון:**
- מה התדירות האופטימלית?
- איך לטפל בפערים (gaps) באימון?
- איך לחזור לאימון אחרי הפסקה?

## דוגמאות קוד נוכחי

### חישוב ציון Go/No-Go (נוכחי):
```javascript
// משקל: דיוק(55%) + מהירות(30%) + יציבות(15%)
var acc = metrics.accuracy || 0;
var meanRT = metrics.mean_rt_ms || 0;
var cv = metrics.rt_cv || 1;

var speed01 = (max_rt_ms - meanRT) / (max_rt_ms - min_rt_ms);
var stab01 = 1 - (cv / cv_target);
var total01 = acc * 0.55 + speed01 * 0.30 + stab01 * 0.15;
var gameScore = Math.round(total01 * 100);
```

### חישוב ציון Stroop (נוכחי - פשוט מדי):
```javascript
var gameScore = Math.round((metrics.accuracy || 0) * 100);
```

### חישוב ציון יומי (נוכחי):
```php
// ממוצע משוקלל של כל הניסיונות ביום
$total_weighted_score = 0;
$total_weight = 0;
foreach ($attempts as $attempt) {
  $weight = HB_COG_GAME_WEIGHTS[$attempt['game_id']] ?? 1.0;
  $score = $attempt['scores']['game_score'] ?? 0;
  $total_weighted_score += $score * $weight;
  $total_weight += $weight;
}
$daily_score = $total_weight > 0 ? round($total_weighted_score / $total_weight) : 0;
```
