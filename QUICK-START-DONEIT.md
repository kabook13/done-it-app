# 🚀 Quick Start - Done-It

## מה אתה צריך לעשות עכשיו - 3 שלבים פשוטים:

---

## ✅ שלב 1: צור אייקונים (2 דקות)

1. פתח את `icon-generator.html` בדפדפן
2. לחץ "יצירת Icon 192x192" → הקובץ יורד
3. לחץ "יצירת Icon 512x512" → הקובץ יורד
4. העתק את שני הקבצים (`icon-192.png` ו-`icon-512.png`) לאותה תיקייה שבה נמצא `lumina-vault.html`

**✅ סיימת? יש לך 5 קבצים בתיקייה:**
- lumina-vault.html
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png

---

## ✅ שלב 2: העלה לשרת (אם יש לך)

### אופציה A: יש לך שרת (cPanel, FTP וכו')
1. העלה את כל 5 הקבצים לאותה תיקייה בשרת
2. פתח בדפדפן: `https://האתר-שלך.com/שם-תיקייה/lumina-vault.html`

### אופציה B: אין לך שרת
- פתח את `lumina-vault.html` ישירות מהמחשב (לחץ כפול)
- ⚠️ זה יעבוד, אבל לא תוכל להתקין על הטלפון בלי שרת

---

## ✅ שלב 3: התקן על הטלפון (אם יש לך שרת)

### Android (Chrome):
1. פתח את האפליקציה בדפדפן Chrome
2. תפריט (⋮) → "הוסף למסך הבית"
3. ✅ מותקן!

### iPhone (Safari):
1. פתח את האפליקציה בדפדפן Safari
2. Share (□↑) → "הוסף למסך הבית"
3. ✅ מותקן!

---

## 🎯 סיכום מהיר:

```
1. פתח icon-generator.html → צור אייקונים → העתק לתיקייה
2. העלה את כל הקבצים לשרת (או פתח מהמחשב)
3. התקן על הטלפון (אם יש שרת)
```

**זה הכל!** 🎉

## ❓ אין לך שרת? אפשרויות חינמיות:

- **GitHub Pages**: העלה את הקבצים ל-GitHub repository
- **Netlify**: גרור את התיקייה ל-Netlify
- **Vercel**: העלה את הקבצים ל-Vercel

כל אחד מהם ייתן לך כתובת HTTPS חינמית!
