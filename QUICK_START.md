# ⚡ התחלה מהירה - higayonbarie.co.il

## 🎯 5 דקות להתחלה

### שלב 1: פתיחת הפרויקט (1 דקה)

```bash
# פתח Cursor
File → Open Folder → F:\cursor\files\higayonbarie-site
```

### שלב 2: הכרת המבנה (2 דקות)

**תיקיות עיקריות:**
- `plugins/` - כל הפלאגינים המותאמים
- `themes/` - כל ה-Themes המותאמים
- `docs/` - תיעוד מפורט

**קבצים חשובים:**
- `README.md` - סקירה כללית
- `WORKFLOW.md` - תהליך עבודה
- `PROJECT_STRUCTURE.md` - מבנה מפורט

### שלב 3: עבודה על פלאגין (2 דקות)

**דוגמה: עריכת goodlife-wordle**

1. פתח: `plugins/goodlife-wordle/wordle.php`
2. ערוך את הקוד
3. שמור
4. בדוק

### שלב 4: עבודה על Theme (2 דקות)

**דוגמה: עריכת עיצוב**

1. פתח: `themes/hello-theme-child/style.css`
2. ערוך את ה-CSS
3. שמור
4. בדוק
