# פרוטוטייפים - הוראות שימוש

## קבצים שנוצרו

1. **home-v3.html** - עמוד הבית הראשי
2. **start-here.html** - עמוד התחלה עם מסלולים
3. **training-hub.html** - מסלול החשיבה המודרך

## איך לפתוח

### בדפדפן (דסקטופ)
1. פתחו את הקובץ `home-v3.html` ישירות בדפדפן (כפול קליק או גרירה)
2. או פתחו דרך VS Code: לחצו ימין על הקובץ → "Open with Live Server" (אם מותקן)
3. או פתחו דרך שורת הפקודה:
   ```
   start home-v3.html
   ```

### בדפדפן מובייל
**אפשרות 1 - דרך רשת מקומית:**
1. פתחו שרת מקומי (לדוגמה: `python -m http.server 8000` או `npx http-server`)
2. גשו מהטלפון לכתובת: `http://[כתובת-IP-שלכם]:8000/prototypes/home-v3.html`

**אפשרות 2 - DevTools:**
1. פתחו את הקובץ בדפדפן Chrome/Edge
2. לחצו F12 לפתיחת DevTools
3. לחצו על אייקון הטלפון (Toggle device toolbar) או Ctrl+Shift+M
4. בחרו מכשיר מהרשימה (iPhone, Galaxy, וכו')
5. בדקו את התצוגה והתנהגות

## תכונות לבדיקה

### Toggle A/B
- לחצו על הכפתור "גרסה A" בפינה השמאלית העליונה
- זה יחליף בין שתי גרסאות (שינויי ניסוח והיררכיה קלים)

### רספונסיביות
- בדקו בגדלים שונים: דסקטופ, טאבלט, מובייל
- כל הכפתורים והכרטיסים אמורים להתאים את עצמם

### אינטראקטיביות
- כפתור "לקרוא עוד" בהמלצות (home-v3.html)
- Tabs ברמות (training-hub.html)
- כל הקישורים בין העמודים

## הערות טכניות

- כל הקבצים הם עצמאיים (CSS ו-JS פנימיים)
- אין תלויות חיצוניות
- RTL מלא (direction: rtl)
- עיצוב מודרני עם gradients ו-shadows
- רספונסיבי מלא למובייל
