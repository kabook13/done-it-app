# 🤖 עבודה עם AI - מדריך מעשי

## 🎯 מבוא

מדריך זה מסביר איך לעבוד עם AI (Cursor/Auto) על פיצ'רים חדשים בפרויקט בצורה מסודרת ויעילה.

---

## 🚀 התחלת שיחה חדשה

### שלב 1: פתיחת שיחה

**ב-Cursor:**
1. לחץ על `Ctrl+L` (או `Cmd+L` ב-Mac) לפתיחת Chat
2. או לחץ על כפתור Chat בתפריט

**ב-ChatGPT/Claude:**
1. פתח שיחה חדשה
2. התחל עם הקשר על הפרויקט

---

## 📋 איך להציג את הקונטקסט

### שיטה 1: פתיחת קבצים רלוונטיים (מומלץ!)

**לפני שאתה מתחיל שיחה, פתח את הקבצים הבאים:**

1. **המדריך הראשי:**
   ```
   START_HERE.md
   ```

2. **מפת העבודה:**
   ```
   docs/FEATURE_ROADMAP.md
   ```

3. **מדריך יצירת פיצ'רים:**
   ```
   docs/CREATING_NEW_FEATURES.md
   ```

4. **פלאגין דוגמה (אם רלוונטי):**
   ```
   plugins/goodlife-wordle/wordle.php
   ```

**למה?** כש-AI רואה את הקבצים הפתוחים, הוא מבין את הקונטקסט!

---

### שיטה 2: הודעה ראשונית מפורטת

**כשאתה פותח שיחה חדשה, כתוב משהו כזה:**

```
אני עובד על פרויקט higayonbarie.co.il - אתר תשבצי היגיון.

אני רוצה ליצור פיצ'ר חדש: [תיאור הפיצ'ר]

הפרויקט נמצא ב: F:\cursor\files\higayonbarie-site

יש לי מדריכים:
- START_HERE.md - התחלה מהירה
- docs/CREATING_NEW_FEATURES.md - איך ליצור פיצ'רים
- docs/FEATURE_ROADMAP.md - מפת עבודה

אני רוצה שתעזור לי ליצור [שם הפיצ'ר] לפי המבנה והמדריכים הקיימים.
```

**בשיחה, תגיד:**

```
קרא את docs/FEATURE_ROADMAP.md כדי להבין את התכנון
```

### שיטה 2: תיאור המבנה

**אם AI לא רואה את הקבצים, תאר:**

```
הפרויקט שלי:
- plugins/ - כל הפלאגינים המותאמים
- themes/ - כל ה-themes
- docs/ - תיעוד מפורט

הפלאגינים הקיימים (כמו goodlife-wordle) הם:
- קובץ PHP אחד עם shortcode
- CSS inline
- JavaScript inline
- נתונים ב-JSON (לפעמים)

אני רוצה ליצור פלאגין חדש באותו סגנון.
```

## 🔄 תהליך עבודה מומלץ

### שלב 1: תכנון (עם AI)

**תגיד ל-AI:**

```
אני רוצה ליצור [שם פיצ'ר].

בוא נתכנן את זה יחד:
1. מה המבנה?
2. מה הקבצים?
3. מה התכונות?
```

**AI יעזור לך:**
- לתכנן את המבנה
- להחליט על הקבצים
- לתכנן את התכונות

---

### שלב 2: יצירת מבנה (עם AI)

**תגיד ל-AI:**

```
עכשיו בוא ניצור את המבנה:
1. צור תיקייה plugins/[שם-פלאגין]
2. צור קובץ ראשי עם תבנית בסיסית
```

**AI יעזור לך:**
- ליצור את המבנה
- לכתוב תבנית בסיסית
- להגדיר shortcode

---

### שלב 3: פיתוח (עם AI)

**תגיד ל-AI:**

```
עכשיו בוא נוסיף:
1. שאלות/נתונים
2. HTML/CSS לממשק
3. JavaScript לאינטראקטיביות
4. לוגיקה
```

**AI יעזור לך:**
- לכתוב קוד
- לפתור בעיות
- לשפר ולעדכן

---

### שלב 4: בדיקה (עם AI)

**תגיד ל-AI:**

```
בוא נבדוק:
1. יש שגיאות?
2. הכל עובד?
3. צריך לשפר משהו?
```

**AI יעזור לך:**
- למצוא שגיאות
- לבדוק קוד
- לשפר

---

## 💡 טיפים לעבודה יעילה

### 1. פתח קבצים רלוונטיים

**לפני שיחה, פתח:**
- המדריכים הרלוונטיים
- פלאגין דוגמה (אם יש)
- קבצים שאתה עובד עליהם

---

### 2. תן הקשר ברור

**תמיד תגיד:**
- מה אתה רוצה לעשות
- איפה הפרויקט
- מה המבנה הקיים
- מה הסגנון

---

### 3. עבוד שלב אחר שלב

**אל תבקש הכל בבת אחת:**
- תכנן קודם
- צור מבנה
- הוסף תכונות בהדרגה
- בדוק תוך כדי

---

### 4. השתמש במדריכים

**תמיד הפנה ל-AI למדריכים:**
- "קרא את START_HERE.md"
- "השתמש בתבנית מ-CREATING_NEW_FEATURES.md"
- "עקוב אחרי FEATURE_ROADMAP.md"

---

### 5. בקש הסברים

**אם משהו לא ברור:**
- "תסביר לי איך זה עובד"
- "למה עשית ככה?"
- "איך זה מתחבר לשאר?"
