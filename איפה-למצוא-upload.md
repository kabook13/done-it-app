# 📍 איפה למצוא "uploading an existing file"?

## 🎯 אחרי שיצרת Repository חדש:

### שלב 1: יצירת Repository
1. היכנס ל-GitHub → לחץ **"New"** (כפתור ירוק) או ה-`+` בפינה הימנית העליונה
2. בחר **"New repository"**
3. מלא:
   - שם: `done-it-app`
   - **Public** ✅
   - **אל תסמן** "Add a README file"
   - **אל תסמן** "Add .gitignore"
   - **אל תסמן** "Choose a license"
4. לחץ **"Create repository"**

### שלב 2: מציאת הכפתור

**אחרי שיצרת את ה-repository, תראה עמוד עם הודעות:**

```
Quick setup — if you've done this kind of thing before
  [קוד Git...]

or

  [קישור ל-SSH...]

or

  [קישור ל-HTTPS...]

or

  ⬇️ upload an existing file
```

**הכפתור נמצא למטה!** 👇

---

## 🔍 אם אתה לא רואה את הכפתור:

### אופציה A: Repository לא ריק
אם הוספת README או קבצים אחרים:
1. לחץ על **"Add file"** (כפתור ירוק בפינה הימנית העליונה)
2. בחר **"Upload files"**

### אופציה B: דרך Add file
1. בפינה הימנית העליונה של ה-repository, לחץ על הכפתור הירוק **"Add file"**
2. בחר **"Upload files"**
3. גרור את הקבצים או לחץ **"choose your files"**

## ✅ דרך חלופית (אם עדיין לא מוצא):

### דרך Add file:
1. בפינה הימנית העליונה, לחץ על הכפתור הירוק **"Add file"**
2. תפריט נפתח → בחר **"Upload files"**
3. עכשיו תראה:
   - אזור לגרירת קבצים
   - או כפתור **"choose your files"**

---

## 🎯 סיכום - 2 דרכים:

### דרך 1: אחרי יצירת Repository ריק
- גלול למטה → תראה **"upload an existing file"**

### דרך 2: דרך Add file
- לחץ **"Add file"** (כפתור ירוק) → **"Upload files"**

---

## 💡 טיפ:

אם אתה לא רואה את הכפתור, זה אומר שה-repository כבר לא ריק.
פשוט השתמש ב-**"Add file"** → **"Upload files"** - זה אותו דבר!
