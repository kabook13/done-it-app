## 📤 שלב 1: עדכון הקובץ ב-GitHub

### א. פתיחת הקובץ לעריכה:

1. **פתח את GitHub** → היכנס ל-account שלך
2. **פתח את ה-repository** `done-it-app` (או השם שלך)
3. **לחץ על הקובץ** `lumina-vault.html`
4. **לחץ על כפתור העיפרון** (✏️) בפינה הימנית העליונה
   - זה יפתח את הקובץ לעריכה

### ב. העתקת התוכן החדש:

1. **פתח את הקובץ** `lumina-vault.html` במחשב שלך
   - זה הקובץ בתיקייה: `f:\cursor\files\higayonbarie-site\lumina-vault.html`
2. **העתק את כל התוכן** (Ctrl+A, Ctrl+C)
3. **חזור ל-GitHub**
4. **הדבק את התוכן** ב-GitHub (Ctrl+V)
   - זה יחליף את כל התוכן הישן

### ג. שמירה:

1. **גלול למטה** עד שתגיע לכפתור "Commit changes"
2. **בשדה "Commit message"** כתוב: `עדכון: הוספת מתנות ממני ושינוי סיסמה`
3. **לחץ על "Commit changes"** (כפתור ירוק)
4. **חכה 1-2 דקות** → GitHub יעדכן את האתר

## 📱 שלב 3: התקנה בטלפון

### א. פתיחת האפליקציה בטלפון:

1. **פתח דפדפן בטלפון** (Chrome, Safari)
2. **גש לכתובת:**
   ```
   https://YOUR_USERNAME.github.io/done-it-app/lumina-vault.html
   ```
   (אותו שם משתמש כמו קודם)

### ב. התקנה כאפליקציה (PWA):

#### **אם אתה משתמש ב-iPhone (Safari):**

1. **לחץ על כפתור "Share"** (הכפתור עם החץ למעלה) בתחתית המסך
2. **גלול למטה** → לחץ על **"הוסף למסך הבית"** (Add to Home Screen)
3. **לחץ "הוסף"** בפינה הימנית העליונה
4. **האפליקציה תופיע** על מסך הבית עם האייקון שלך! 🎉

#### **אם אתה משתמש ב-Android (Chrome):**

1. **לחץ על תפריט** (3 נקודות בפינה הימנית העליונה)
2. **לחץ על "הוסף למסך הבית"** (Add to Home Screen)
   - או תראה הודעה: "הוסף Done-It למסך הבית"
3. **לחץ "הוסף"** או "Install"
4. **האפליקציה תופיע** על מסך הבית! 🎉
